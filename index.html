This project is a flutter web game built with flutter's flame game engine following a tutorial courtesy of google. Check it out here: https://eleisian.github.io/brick-breaker/

This project is a rock paper scissors computer that you can play in a terminal. It was creating with the jQuery-terminal library. By inputting your own API code you can also talk to a tuned version of ChatGPT that responds slightly offensively if you input wrong commands. You can try it out here.

This project is a proof of concept for creating 3D scenes with the react-three-fiber library. FBX objects were imported and textures were applied. These loading screens were created with Blender and Adobe After Effects. Theoretically these loading screens can be linked with a scripting language to be placed within a website or video game. Check it out here: https://eleisian.github.io/loading/

This project was built upon an existing project that I found on Codesandbox.io Someone had already created an Ascii-renderer for 3D objects with react-three-fiber. However they did not add a function so that you can import your own GLTF objects to the scene. I just created a function to do so. https://eleisian.github.io/ascii-renderer

This project is a water shader created with procedural Unity nodes. Click on the windmill icon to proceed to the next screen. Hopefully you find that each iteration the scene gets better. https://eleisian.github.io/everywhere-is-the-ocean
